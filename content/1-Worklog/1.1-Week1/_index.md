### Week 1 Achievements:

* Foundational understanding of the AWS ecosystem:  
  * Know how to create an account
  * Set up cost management alerts (AWS Budgets)
  * Support (AWS Support) 
  * Manage secure access via IAM

* Familiarized with the AWS Management Console and know how to find, access, and use services from the web interface

* Basic knowledge and skills about networking (VPC):
  * Know the concept of Virtual Private Cloud (VPC)
  * Understand and configure how data flows through Internet Gateway and NAT Gateway

* Cloud server management (Basic Amazon EC2):

  * Understand EC2 instance types
  * Server images (AMI)
  * Storage services (EBS)
