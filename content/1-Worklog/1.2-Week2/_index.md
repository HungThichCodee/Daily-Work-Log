### Week 2 Objectives:

* Learn about AWS Cloud9 and how to host a website using Amazon S3.
* Understand relational database concepts with Amazon RDS.
* Learn about Amazon Lightsail for quick and simple application deployment.
* Learn about deploying applications using Docker on Amazon Lightsail Container.
* Learn about Amazon CloudWatch.

### Week 2 Achievements:

* Development Environment & Storage (Cloud9, S3): 
  * Understand Cloud9 and S3.
  * Understand and successfully practice deploying, hosting a static website through Amazon S3 service

* Amazon RDS:
  * Grasp core knowledge and how to operate a fully managed relational database with Amazon RDS service

* Amazon Lightsail:
  * Grasp the concept and know how to use Amazon Lightsail service to quickly create, manage, and deploy applications.

* Docker & Lightsail Container:
  * Grasp core knowledge and how to operate an application container with Docker
  * Deploy applications using Docker on Amazon Lightsail Container.

* Amazon CloudWatch:
  * Grasp core knowledge about Metric, Log, Alarm concepts.
  * Practice creating dashboards and setting up alerts.
